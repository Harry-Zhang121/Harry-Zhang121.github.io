Constructor and Destructor

Constructor is a function that called automaticly right after the creation of the class.
Constructor must have the *same name as the class, and must not have return type.
Destructor is a special class method which called automatically when object is released. 

class point
{
    private:
        int x,y;
    public:
    point (int nx, int ny){x=nx;y=ny;}    //constructor with multiple arguments
    point (int nx){x=nx; y=0;}            //conversion constructor
    point () {x=0; y=0;}                //defult constructor
    ~point(/*must be empty*/) {/*body*/} // destructor
    point(const point &p) {x=p.x; y=p.y;}//copy constructor
}
